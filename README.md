# psniffer
Packet Analysis Tool

**Installation**

psniffer: 
```
$ make
```
Client:
```
$ make client
```


**Usage**

psniffer:
```
$ ./psniffer <interface>
```


client:
```
$ ./client <command>
```
Supported commands for client program:

- *close* : closes the program and displays stats
- *print* : print stats
- *clear* : free all memory and clear hash table
